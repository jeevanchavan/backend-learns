# InstaClone Backend Service

## 1. Introduction
InstaClone is a backend service that provides REST APIs for a social media platform similar to Instagram.  
The system allows users to create accounts, upload posts, like posts, comment, and interact with other users.

This project is built to understand real-world backend architecture including authentication, database design, and API handling.

---

## 2. Objective
The objective of this project is:

- To design a scalable backend for a social media application
- To implement user authentication and authorization
- To manage posts, likes, comments and follow system
- To learn database schema design
- To understand API design and error handling

---

## 3. Target Users
- General users who want to share posts and photos
- Users who want to interact (like, comment, follow)

---

## 4. Tech Stack

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose ODM

### Authentication
- JWT (JSON Web Token)
- Cookies / Token based auth

### Tools Used
- Postman (API testing)
- Git & GitHub (version control)

---

## 5. Core Functionalities

The backend provides APIs for:

- User Registration
- User Login
- Authentication & Authorization
- Create Post
- Delete Post
- Like / Unlike Post
- Comment on Post
- Follow / Unfollow Users
- Fetch User Feed

---

## 6. System Type
This is a REST API based backend service.  
The frontend (React app) communicates with this backend using HTTP requests.

---
